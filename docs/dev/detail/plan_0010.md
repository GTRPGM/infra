# plan_0010 - 테스터 컨텍스트 동기화 및 정합성 어설트 강화

## 상태
- 완료 (2026-02-07)

## 목표
- 테스터가 GM이 참조하는 컨텍스트를 동일하게 조회하도록 정렬한다.
- 턴 종료 시점에 관계 변화/시퀀스 전이/아이템 획득 정합성을 자동 어설트한다.

## 범위
- 수정 대상: `tester/src/tester/*`, `tests/*`, `docs/dev/*`
- 비대상: `rule-engine`, `state-manager` 직접 수정

## 작업 단위
1. 컨텍스트 조회 정렬
- GM 기준 조회 경로(`session`, `sequence/details`, `act/details`)를 테스터에 동일 반영
- 보조 검증용(`player`, `inventory`, `npcs`, `enemies`)도 함께 수집

2. 정합성 어설트 추가
- 세션/시퀀스/액트 ID 상호 일치 어설트
- 관계(`entity_relations`, `player_npc_relations`) 참조 무결성 어설트
- 시퀀스 변경 시 canonical ID/시나리오 명세 포함 여부 어설트
- “획득”류 서술 발생 시 inventory 증가 어설트(휴리스틱)

3. 회귀 테스트 보강
- 기존 runner 회귀 테스트를 신규 컨텍스트 구조에 맞게 보정
- 정합성 어설트 전용 실패/성공 테스트 추가

## 완료 기준 (DoD)
- 테스터 상태 스냅샷에 GM 컨텍스트가 포함되어야 함.
- 턴별 정합성 위반이 자동 실패로 감지되어야 함.
- `uv` 테스트에서 기존 회귀 + 신규 어설트 테스트 통과.
