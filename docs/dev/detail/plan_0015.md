# plan_0015 - 테스터 실행 및 시퀀스/상태 업데이트 정밀 분석

## 상태
- 진행 중 (2026-02-08)

## 목표
- 통합 테스터를 실행해 시나리오 진행이 기능적으로 중단 없이 수행되는지 확인한다.
- 시퀀스별 등장/퇴장 엔티티 전이와 시퀀스 타입(탐색/상호작용/전투) 진행 정합성을 검증한다.
- 아이템/엔티티(특히 적 HP, 목록 변화) 상태 업데이트가 턴별로 반영되는지 분석한다.

## 작업 단위
1. 실행 준비
- 러너 프로파일/로그 포맷 점검
- 실행 대상: `three_sequence_combat` 프로파일

2. 테스터 실행
- `PYTHONPATH=tester/src uv run python -m tester.runner <session_id> <max_turns> three_sequence_combat`
- 최소 1회 이상 실행 후 로그 파일 확보

3. 로그 정밀 분석
- 시퀀스 전이 이벤트(`seq-1 -> seq-2 -> seq-3`) 검출
- 전이 시 등장/퇴장 엔티티 검증
- 시퀀스 타입별 진행 증거(목표/트리거/내러티브 패턴) 수집
- 상태 업데이트(적 HP 감소, 적 목록 소거, 인벤토리 변화 여부) 추적

4. 결과 문서화
- 실행 결과와 판단 근거를 `docs/dev/detail` 문서에 기록
- 미흡점/잠재 리스크/후속 보완 항목 정리

## 완료 기준 (DoD)
- 테스터 실행 로그가 생성되고 오류 중단 여부가 확인된다.
- 시퀀스별 등장/퇴장 및 타입 진행 분석이 근거 라인과 함께 정리된다.
- 아이템/엔티티 상태 업데이트 분석 결과가 명시된다.
