# plan_0017 - 3액트/6시퀀스 고도화 평가 시나리오 및 인벤토리 이슈 분석

## 상태
- 완료 (2026-02-08)

## 목표
- 3개 액트, 6개 시퀀스 구성의 고도화 평가 시나리오를 생성한다.
- 필수 아이템 5종 + 플레이어 시작 장비(포션, 무기)를 반영한다.
- NPC 5종(각 최소 2개 아이템 연관), 적 5종(각 최소 2개 아이템 연관)을 반영한다.
- 시나리오 주입 외 테스트 흐름은 BE-router 경유로 실행한다.
- 테스트 후 아이템/인벤토리 미반영 원인을 코드/로그 기반으로 분석한다.

## 작업 단위
1. 시나리오/프로파일 자산 추가
- 신규 payload/profile 추가
- 프로파일 로더 등록

2. 시작 인벤토리 반영 경로 추가
- BE-router 경유로 세션 시작 후 플레이어 인벤토리에 포션/무기 지급

3. 주입 및 테스트 실행
- `seed_profile`로 시나리오 주입
- `tester.runner`로 BE-router 경유 턴 테스트

4. 분석 문서화
- 전이/상태 업데이트 결과 정리
- 아이템/인벤토리 미반영 원인 분석

## 완료 기준 (DoD)
- 요구 스펙(3액트/6시퀀스/5아이템/5NPC/5적/시작 장비) 시나리오가 주입됨
- BE-router 경유 테스트 로그가 확보됨
- 인벤토리 이슈 원인이 문서로 정리됨
